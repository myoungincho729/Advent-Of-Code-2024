# Advent-Of-Code-2024

### command-line rule
```
arg1: {problem_number}-{part_number}
arg2:
    empty -> puzzle input
    not empty -> test input name

example:
    cargo run 7-2
    -> run problem 7, part 2, puzzle input, filename: input/d7.txt
    cargo run 7-1 t1
    -> run problem 7, part 1, test input 1, filename: input/d7t1.txt
```
